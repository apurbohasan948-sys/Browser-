# 📱 GitHub APK Build & Release Guide — AuraMini AI Browser 

গিটহাবের (GitHub Actions) মাধ্যমে কীভাবে ওয়ান-ক্লিক বা অটোমেটিক উপায়ে অ্যান্ড্রয়েড **AuraMini AI Browser**-এর সচল APK বিল্ড এবং ডাউনলোড করবেন, তার জন্য এই সহজ নির্দেশিকা তৈরি করা হয়েছে। 

নিচে ধাপে ধাপে সম্পূর্ণ প্রক্রিয়া বাংলায় ব্যাখ্যা করা হলো।

---

## 🛠️১. মূল ফিচারসমূহ (GitHub Integration Features)
আমরা এই প্রজেক্টে একটি আধুনিক **CI/CD Build Pipeline** যুক্ত করেছি:
* **অটো-বিল্ড (Auto-Build on Push):** প্রতিবার প্রজেক্টের মেইন কোড গিটহাবে পুশ করার সাথে সাথে সেটি স্বয়ংক্রিয়ভাবে একটি নতুন APK বিল্ড রেডি করে দিবে।
* **ম্যানুয়াল ট্রিগার (Manual Dispatch):** আপনি চাইলে গিটহাবের ড্যাশবোর্ড থেকে যেকোন সময় বোতাম টিপে ম্যানুয়ালি বিল্ড সম্পন্ন করতে পারবেন।
* **বিল্ড আর্টিফ্যাক্ট (Artifacts Output):** সফলভাবে বিল্ড শেষ হবার পর সহজেই মেইন ব্রাউজার ড্যাশবোর্ড থেকে রেডি-টু-ইনস্টল `.apk` ডাউনলোড করা যাবে।

---

## 🚀 ২. বিল্ড করার পর্যায়ক্রমিক ধাপসমূহ (Step-by-Step Guide)

### ধাপ ১: প্রজেক্টটি গিটহাবে আপলোড করুন (Push to GitHub)
প্রথমেই আপনার লোকাল কম্পিউটার বা ক্লাউড থেকে প্রজেক্টটিকে একটি গিটহাব রিপোজিটরিতে (GitHub Repository) পুশ করতে হবে। রিপোজিটরিটি চাইলে **Public** অথবা সম্পূর্ণ **Private** হতে পারে।
```bash
git init
git add .
git commit -m "Initialize AuraMini with Auto-APK Build pipeline"
git branch -M main
git remote add origin https://github.com/your-username/your-repo-name.git
git push -u origin main
```

### ধাপ ২: গিটহাব অ্যাকশন্স ট্যাবে যান (Navigate to GitHub Actions)
1. আপনার ব্রাউজারে তৈরি করা GitHub repository-টি ওপেন করুন।
2. সাব-হেডিং মেনু বার থেকে **Actions** টেক্সট আইকনটি সিলেক্ট করুন।
3. বামদিকের মেনুতে **"Build AuraMini APK"** নামে আমাদের ডেডিকেটেড কাজের ফ্রেমওয়ার্ক দেখতে পাবেন।

### ধাপ ৩: রান করুন বা বিল্ড স্ট্যাটাস দেখুন (Run Workflow or Monitor Build)
* **পুশ করার পর:** আপনি কোড পুশ করার পর এটি অটোমেটিক রান করতে শুরু করবে। আপনি বিল্ডটি ক্লিক করে স্ক্রিনে লাইভ প্রোগ্রেস (`Set up JDK`, `Setup Gradle`, `Build Debug APK`) দেখতে পাবেন।
* **ম্যানুয়াল রান করতে চাইলে:**
  1. বামে **Build AuraMini APK** এ ক্লিক করুন।
  2. ডানে একটি হালকা ড্রপডাউন বাটন পাবেন, যেখানে লেখা থাকবে **"Run workflow"**।
  3. বাটনটি ক্লিক করুন এবং সবুজ রান প্রোফাইলে টিক চিহ্ন আসার অপেক্ষা করুন।

### ধাপ ৪: APK ডাউনলোড করুন (Download the Compiled APK)
1. বিল্ডটি সফলভাবে শেষ হলে (সবুজ চেকমার্ক আসার পর) বিল্ড রানিং অপশনটিতে ক্লিক করুন।
2. স্ক্রিন স্ক্রোল করে নিচের দিকে **Artifacts** সেকশনে চলে যান।
3. ওখানে **`auramini-debug-apk`** নামের একটি ডাউনলোড লিংক দেখতে পাবেন।
4. ফাইলটি ক্লিক করলেই একটি জিপ ফাইল আকারে আপনার তৈরি সচল অ্যাপ ডাউনলোড হয়ে যাবে। এটি আনজিপ করলেই পেয়ে যাবেন অ্যান্ড্রয়েডে ইনস্টল করার উপযোগী ফাইনাল `app-debug.apk` ফাইলটি!

## 🔐 ৩. সিক্রেট কী ম্যানেজমেন্ট (Secure Secrets Configuration)

অ্যান্ড্রয়েড অ্যাপে সরাসরি কোডের ভেতরে **API Key** বা সংবেদনশীল ডেটা হার্ডকোড করা অত্যন্ত ঝুঁকিপূর্ণ। গিটহাবে এই বিষয়টি নিরাপদে হ্যান্ডেল করতে নিচের পদ্ধতিটি অনুসরণ করুন:

1. আপনার গিটহাব রিপোজিটরির **Settings** ট্যাবে যান।
2. বামদিকের প্যানেল থেকে **Secrets and variables** > **Actions** সিলেক্ট করুন।
3. **"New repository secret"** বাটনে ক্লিক করুন।
4. নিচের কী-টি অ্যাড করুন:
   * **Name:** `GEMINI_API_KEY`
   * **Value:** আপনার রিয়াল Google AI Studio থেকে সংগ্রহ করা Gemini API Key (যা দিয়ে ব্রাউজারের এআই ফিচার সচল হবে)।

আমাদের কোডবেস সরাসরি ডায়নামিক এনভায়রনমেন্ট কনফিগারেশনের সাথে লিংক করা, ফলে আপনার সিক্রেট প্রপার্টিজগুলো সম্পূর্ণ ক্লাউড স্যান্ডবক্সে প্রোটেক্টেড থাকবে।
